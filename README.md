# scoop-bucket

A [Scoop](https://scoop.sh) bucket for Sworld's projects.

## Available

| App | Install |
|-----|---------|
| easy-nats | `scoop install easy-nats` |

## Usage

```powershell
scoop bucket add sworld https://github.com/mcthesw/scoop-bucket
scoop install <app>
```

Manifests are updated automatically on each release.
